## Getting Started

This project created for about Server Side Rendering with React.

### Installing

First clone project and install dependencies

```sh
$ mkdir demo && cd demo
$ git clone https://github.com/PriyadarshiniPuja/demo.git
$ cd demo
$ npm install
```

Run on local

```sh
$ npm run dev
```

Navigate to [http://localhost:3000](http://localhost:3000)

## Run test cases

\$ npm run test
